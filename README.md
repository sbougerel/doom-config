# My Doom Emacs Setup

## Installation

### Archlinux

```bash
yay -Sy emacs-nativecomp
git clone https://github.com/sbougerel/doom-config.git ~/.doom.d
git clone --depth 1 https://github.com/doomemacs/doomemacs ~/.emacs.d
~/.emacs.d/bin/doom install
```

Add `$HOME/.emacs.d/bin` to $PATH
